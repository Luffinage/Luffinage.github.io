Навыки

Что я использую в работе

HTML5

Именно он создает каркас вашего сайта или приложения, а пятая версия позволит мне создавать более SEO-оптимизированную структуру вашего продукта

CSS3

Этот язык стилей позволяет мне создавать абсолютно любой внешний вид вашего сайта или приложения. Все ограничивается только вашей фантазией!

Java Script

Этот язык программирования позволяет оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое

Sass

Метаязык, который упрощает и ускоряет написание CSS-кода. Его часто называют препроцессором CSS.

jquery

Библиотека Jquery позволит ускорить разработку. Без необходимости интегрировать в проект мы её не будем, но навык работы с ней присутствует

Bootstrap

Фреймворк. Набор инструментов для создания сайтов и веб-приложений. Включает в себя HTML и CSS шаблоны оформления веб-интерфейса, включая JavaScript расширения.

Gulp

Инструмент для автоматизации рутинных задач, которые возникают при веб-разработке.

Git

Система контроля версий с активной поддержкой и открытым исходным кодом, которую используют тысячи разработчиков из разных точек мира.

GitHub

Веб-сервис для хостинга IT-проектов и их совместной разработки.

figma

Графический онлайн-редактор для совместной работы.
